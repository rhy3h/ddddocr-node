    (constant) CHARSET_RANGE :number

    Charset range constants that define different character sets for OCR. These constants represent various combinations of character types (lowercase, uppercase, numeric).
    Properties:
    Name Type Description
    NUM_CASE number Character set containing only numeric characters.
    LOWWER_CASE number Character set containing only lowercase characters.
    UPPER_CASE number Character set containing only uppercase characters.
    MIX_LOWWER_UPPER_CASE number Character set containing both lowercase and uppercase characters.
    MIX_LOWWER_NUM_CASE number Character set containing both lowercase and numeric characters.
    MIX_UPPER_NUM_CASE number Character set containing both uppercase and numeric characters.
    MIX_LOWWER_UPPER_NUM_CASE number Character set containing lowercase, uppercase, and numeric characters.
    NO_LOWEER_UPPER_NUM_CASE number Character set containing neither lowercase, uppercase, nor numeric characters.
