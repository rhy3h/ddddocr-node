    setRanges(charsetRange) → {DdddOcr}

    Sets the range restriction for OCR results. This method restricts the characters returned by OCR based on the input: - For `number` input, it applies a predefined character set. Supported values are: - `0`: Digits (0-9) - `1`: Lowercase letters (a-z) - `2`: Uppercase letters (A-Z) - `3`: Lowercase + Uppercase letters - `4`: Lowercase letters + Digits - `5`: Uppercase letters + Digits - `6`: Lowercase + Uppercase letters + Digits - `7`: Default set (a-z, A-Z, 0-9) - For `string` input, each character in the string is treated as a valid OCR result.
